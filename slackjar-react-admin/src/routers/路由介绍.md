# 路由系统介绍

## 📚 技术栈

本项目采用 **React Router v6** 作为路由解决方案，结合以下技术实现完整的路由管理：

- **react-router**: 核心路由库
- **react-router-dom**: DOM 绑定，提供 BrowserRouter 等组件
- **懒加载**: React.lazy + Suspense 实现代码分割
- **权限控制**: 自定义守卫组件（CheckLogin、CheckPerms）

---

## 🔧 核心实现

### 1. 路由类型定义

**文件**: `src/routers/router.tsx`

```typescript
/**
 * 扩展 RouteObject，增加菜单和图标属性
 */
export declare type MenuRouteObject = {
    icon?: React.ReactNode;      // 菜单图标
    label?: string;              // 菜单显示文本（用于国际化）
    children?: MenuRouteObject[] | null;  // 子路由/子菜单
} & RouteObject;
```

**说明**：
- 继承 React Router 的 `RouteObject` 类型
- 扩展 `icon`、`label`、`children` 属性，支持菜单生成
- `label` 使用国际化键名，如 `"menu.home"`

### 2. 路由模块定义

#### 示例：欢迎页模块

**文件**: `src/routers/modules/welcome.tsx`

```typescript
import {HomeOutlined} from "@ant-design/icons";
import lazyLoad from "../lazyLoad";
import React, {lazy} from "react";
import {MenuRouteObject} from "../router";

const welcome: MenuRouteObject = {
    path: "index",
    label: "menu.home",
    icon: <HomeOutlined />,
    element: lazyLoad(lazy(() => import("../../pages/Welcome")))
}

export default welcome
```

**关键点**：
- `path`: 路由路径（相对路径）
- `label`: 国际化键名
- `icon`: Ant Design 图标组件
- `element`: 使用 `lazyLoad` 包装的懒加载组件

### 3. 路由聚合

**文件**: `src/routers/router.tsx`

```typescript
/**
 * 需要登录后才能访问的路由（后台管理路由）
 */
const portalRouters: MenuRouteObject[] = [
    {...welcome},   // 首页/欢迎页
    {...user},      // 用户管理模块
    {...system},    // 系统设置模块
    {...test},      // 测试模块
]

/**
 * 全部路由配置
 */
const routers: MenuRouteObject[] = [
    {
        path: "/",
        // CheckLogin 是登录守卫，未登录会重定向到 /login
        // Portal 是主布局组件（包含头部、侧边栏、内容区）
        element: <CheckLogin><Portal/></CheckLogin>,
        // 404 错误页面
        errorElement: <Error404/>,
        icon: <PieChartOutlined/>,
        // 子路由，对应 portalRouters 中定义的具体页面
        children: portalRouters,
    },
    // 展开不需要登录的路由（登录页等）
    ...front
];

/**
 * 创建浏览器路由实例
 */
const router = createBrowserRouter(routers);
```

**说明**：
- 使用展开运算符 `{...welcome}` 创建路由副本
- 根路由 `/` 包裹 `CheckLogin` 守卫和 `Portal` 布局
- `children` 包含所有需要登录的子路由
- `front` 包含不需要登录的路由（如登录页）

### 4. 懒加载封装

**文件**: `src/routers/lazyLoad.tsx`

```typescript
/**
 * 路由懒加载封装函数
 * 结合 React.lazy 和 Suspense 实现组件按需加载，并提供 Loading 状态
 */
const lazyLoad = (LazyComponent: React.LazyExoticComponent<any>): React.ReactNode => {
    return (
        <Suspense
            fallback={
                <Spin size="large" style={{...}} />
            }
        >
            <LazyComponent/>
        </Suspense>
    );
};
```

**作用**：
- 包裹 `React.lazy()` 组件
- 提供统一的 Loading 状态
- 提升首屏加载性能

## 🎯 核心功能

### 1. 菜单自动生成

**文件**: `src/routers/router.tsx`

```typescript
/**
 * 将路由配置转换为 antd Menu 组件所需的菜单数据结构
 */
const createMenuItems = (routers, key, t, topMenuOnly) => {
    return routers.map(item => ({
        key: key + processPath(item.path),
        icon: item.icon,
        children: !topMenuOnly && item.children 
                  ? createMenuItems(item.children, key + processPath(item.path), t) 
                  : null,
        label: t(item.label || item.path),  // 国际化翻译
    }));
};

/**
 * 获取所有菜单项
 */
const useMenuItems = (topMenuOnly?: boolean): MenuItem[] => {
    const {t} = useTranslation();
    return createMenuItems(routers[0].children, "", t, topMenuOnly);
}
```

**使用**：

```typescript
// 顶部导航菜单
const topMenus = useMenuItems(true);

// 侧边栏子菜单
const childMenus = useChildMenuItems(pathname);
```

### 2. 面包屑导航

```typescript
/**
 * 生成面包屑导航数据
 */
const getBreadcrumbs = (t: Function, pathname: string): string[] => {
    let breadcrumbs: string[] = [];
    let tempRouters: MenuRouteObject[] = routers[0].children!;
    let matchedPath = "";
    
    while (tempRouters) {
        let isMatchedRouters = false;
        for (let router of tempRouters) {
            const routerPath = matchedPath + processPath(router.path);
            if (routerPath === pathname || pathname.startsWith(routerPath + "/")) {
                breadcrumbs.push(t(router.label || router.path!))
                matchedPath = routerPath;
                isMatchedRouters = true;
                tempRouters = router.children as MenuRouteObject[]
                break
            }
        }
        if (!isMatchedRouters) {
            break;
        }
    }
    return breadcrumbs;
}
```

**使用**：

```typescript
const {t, i18n} = useTranslation();
const breadcrumbs = useMemo(() => getBreadcrumbs(t, pathname), 
    [i18n.language, pathname]);

// 结果：["首页", "用户管理", "用户列表"]
```

### 3. 路由路径处理

```typescript
/**
 * 处理路由路径，确保格式正确
 */
const processPath = (path: string | undefined): string => {
    if (path) {
        // 去掉末尾的斜杠
        if (path.lastIndexOf("/") == path.length - 1) 
            path = path.substring(0, path.length - 1);
        // 确保开头有斜杠
        if (path.indexOf("/") != 0) 
            path = "/" + path;
    }
    return path ? path : "";
}
```

---

### 4. 获取父级路由的第一个子路径

```typescript
/**
 * 获取父级路由下的第一个子路由路径
 * 用于点击顶部菜单时自动跳转到第一个子页面
 */
const getFirstChildPathByParent = (pathname: string): string | null => {
    for (let router of routers[0].children!) {
        const searchString = processPath(router.path);
        if (searchString === pathname || pathname.startsWith(searchString + "/")) {
            if (router.children && router.children.length > 0) {
                return searchString + processPath(router.children[0].path);
            }
        }
    }
    return null;
}
```

**使用场景**：
点击顶部"用户管理"菜单时，自动跳转到 `/user/list`（第一个子路由）。

## 🔄 路由工作流程

### 应用启动流程

```
应用启动
    ↓
创建路由实例：createBrowserRouter(routers)
    ↓
RouterProvider 注入路由
    ↓
用户访问 URL（如 /index）
    ↓
React Router 匹配路由
    ↓
CheckLogin 检查登录状态
    ├─ 未登录 → 跳转到 /login
    └─ 已登录 → 继续
    ↓
渲染 Portal 布局
    ├─ MyHeader（顶部导航）
    ├─ MySider（侧边栏菜单）
    └─ Layout.Content
        └─ CheckPerms 检查权限
            ├─ 无权限 → 显示 403
            └─ 有权限 → 渲染 Outlet（页面组件）
```

---

### 路由切换流程

```
用户点击菜单或链接
    ↓
触发导航（navigate 或 Link）
    ↓
React Router 更新 URL
    ↓
匹配新的路由配置
    ↓
懒加载组件（如果首次访问）
    ├─ 显示 Spin 加载动画
    └─ 下载代码 chunk
    ↓
CheckPerms 检查权限
    ↓
渲染新页面组件
    ↓
更新面包屑导航
```

## 📝 最佳实践

### 1. 路由命名规范

```typescript
// ✅ 好：语义化路径
path: "user/list"
path: "user/edit/:id"

// ❌ 不好：无意义路径
path: "u/l"
path: "e/:i"
```

### 2. 懒加载所有页面

```typescript
// ✅ 好：懒加载
element: lazyLoad(lazy(() => import("../../pages/UserList")))

// ❌ 不好：直接导入
import UserList from "../../pages/UserList";
element: <UserList />
```

### 3. 使用国际化键名

```typescript
// ✅ 好：使用 label
label: "menu.user management"

// ❌ 不好：硬编码文本
label: "用户管理"
```

### 4. 合理组织路由模块

```
modules/
├── welcome.tsx      # 首页
├── account.tsx         # 用户相关
├── system.tsx       # 系统设置
└── business.tsx     # 业务模块
```

const user: MenuRouteObject = {
